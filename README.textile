Minimal stateless decision service "Drools":http://www.jboss.org/drools example using Drools 5.2, including a JUnit 4 unit test to run the example.

The example is based on a "Project Euler":http://projecteuler.net/ mathematical problem.

h2. Usage

To run the code, run `src/com/lunatech/euler/test/ProblemsTest.java` as a JUnit test.

h2. Dependencies

The following libraries and JARs are required in the compile-time or run-time classpath.

    drools-5.5.0.Final/antlr-3.3.jar
    drools-5.5.0.Final/antlr-runtime-3.3.jar
    drools-5.5.0.Final/drools-compiler-5.5.0.Final.jar
    drools-5.5.0.Final/drools-core-5.5.0.Final.jar
    drools-5.5.0.Final/knowledge-api-5.5.0.Final.jar
    drools-5.5.0.Final/knowledge-internal-api-5.5.0.Final.jar
    drools-5.5.0.Final/log4j-1.2.14.jar
    drools-5.5.0.Final/mvel2-2.1.3.Final.jar
    drools-5.5.0.Final/slf4j-api-1.6.4.jar
    slf4j-1.7.2/slf4j-log4j12-1.7.2.jar
